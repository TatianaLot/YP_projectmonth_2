# Модель предсказания покупателей на горизонте 90 дней
## Описание проекта
**Цель проекта:** построить модель, определяющую пользователей, которые совершат покупку в ближайшие 90 дней.

## Данные
Использованы исторические данные маркетплейса за период май 2022 — февраль 2024:

- информация о покупках пользователей;
- данные о рекламных коммуникациях;
- целевая переменная — факт покупки в течение 90‑дневного срока.

## Переменные
### Покупки
- число покупок;
- стоимость покупок;
- категории товаров.
### Рекламные сообщения
- канал доставки;
- реакции пользователя (переходы, покупки).
### Созданные признаки
- количество покупок (в целом и по категориям);
- время с момента последней покупки (в целом и по категориям);
- частота покупок;
- длительность взаимодействия с пользователем (от первой до последней покупки);
- число полученных рекламных сообщений;
- количество позитивных действий с рекламными сообщениями (переходы, покупки);
- время с момента последнего сообщения и действие с ним;
- пользовательская конверсия (доля покупок от полученных сообщений).


### Моделирование
* выполнен подбор модели и гиперпараметров с помощью Optuna;
* лучшая модель: **CatBoost**, ROC-AUC = **0.76**.


### Анализ
* проведён анализ важности признаков;
* результаты позволяют выделить ключевые факторы, влияющие на вероятность покупки.
